Inheritance hierarchy for Model

  • ModelFactory
  • Model * ModelValueFactory - create org.ontoware.rdf2go.model.node.* instances * ModelAddRemove - remove statements, diffs, locks * ClosableIterableStatement - read statements * ModelWriter - add statements * Lockable - lock/unlock * ModelRemovePatterns - remove statements matching a given pattern * QueryableModel - * ClosableIterableStatement - what can be queried, can also be read * Sparqlable - SPARQL and other queries * FindableModel - find and count by triple patterns * ModelIO - read/write to/from readers/writers/streams
  • ModelSet * Sparqlable * ModelSetIO * FindableModelSet * ModelSetAddRemove
