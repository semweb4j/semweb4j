RDF2Go API

RDF2o is an implementation-independent Java API with the design goals: portability (hence the name), performance, easy of implementation.

http://semweb4j.org/rdf2go

AIFBcommons collection

AIFBcommons is a small bundle of little, commonly needed tools. Currently, it contains e. .g. iterators such as a ClosableIterator and its corresponding ClosableIterable. Both are used by RDF2Go.

http://aifbcommons-collection
